# Finance Tracker

A full-stack financial tracking system. This project is being developed in versioned stages, with each version introducing a new system capability.

## Goals

- Build a production-style backend architecture
- Design a scalable financial data model
- Implement clean API design
- Practice professional Git workflows
- Deploy a working full-stack application

## Version Roadmap

### v1.0 – Financial Core Engine
- User authentication
- Accounts system
- Transactions system
- Categories
- Budgets
- Live balance calculations
- Docker deployment

### v2.0 – Financial Integrity Engine
- Immutable transaction ledger
- Reversal transactions
- Audit logging
- API versioning

### Future Versions

- Multi-user households
- Bank integrations
- Financial insights engine
- AI recommendations
